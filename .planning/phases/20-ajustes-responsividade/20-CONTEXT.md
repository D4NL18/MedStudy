# Context - Phase 20: Ajustes de Responsividade

## Purpose
A Fase 19 introduziu novos elementos visuais (Badges, Notificações) que foram otimizados para telas desktop (1600px). Esta fase foca na adaptabilidade do sistema para tablets e dispositivos móveis, garantindo que a experiência de estudo seja fluida em qualquer tamanho de tela.

## Scope
- **Dashboard**: Migrar layout de 3 colunas para 1 ou 2 colunas em telas menores.
- **Navbar**: Implementar menu hambúrguer ou barra inferior para mobile.
- **Notificações**: Ajustar dropdown para ocupar largura total ou se transformar em modal no mobile.
- **Tabelas**: Garantir legibilidade dos dados de aulas e revisões.

## Constraints
- Manter a estética Glassmorphism.
- Não quebrar funcionalidades de hover (usar estados ativos alternativos para touch).
