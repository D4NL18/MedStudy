## Module Specifics

### 1. Flashcard Study (Focused Mode)
- **Overlay:** `backdrop-filter: blur(24px) brightness(0.6)`.
- **Card:** `background: rgba(255, 255, 255, 0.03)` with `1px solid rgba(255, 255, 255, 0.1)`.
- **Animation:** 600ms 3D flip on Y-axis with `perspective: 1000px`.
- **Buttons:** Large (64px height) at the bottom for Fácil (Green), Médio (Amber), Difícil (Red).

### 2. Revision Tabs
- **Tabs:** Centralized tabs with sliding underline animation.
- **Badges:** Contagem nas abas (Atrasadas: Vermelho Pulsante, Hoje: Verde Sólido).

### 3. Lesson Plan Priorities
- **Diamond:** Purple/Violet Gradient (`#9D50BB` -> `#6E48AA`) + Glow effect.
- **Alta:** Red (`#EF4444`).
- **Média:** Amber (`#F59E0B`).
- **Baixa:** Emerald/Slate (`#10B981` / `#94A3B8`).

### 4. Markdown Editor
- **Features:** Real-time preview + `Ctrl+V` image paste (Base64 insertion).
