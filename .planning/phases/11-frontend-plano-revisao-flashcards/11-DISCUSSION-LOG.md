## Areas Discussed

### 1. Flashcard Study Experience
- **Options presented:** Focused (overlay), Integrated (page), Mobile-first (large buttons).
- **Selection:** Combination of Focused + Mobile-first.
- **Notes:** Centralized card with dim background and ergonomic buttons at the bottom.

### 2. Revision Layout
- **Options presented:** Tabs, Continuous List, Accordions.
- **Selection:** Tabs.
- **Notes:** One tab per category (Atrasadas, Hoje, Futuras, Realizadas).

### 3. Lesson Plan Priorities
- **Options presented:** Colors/Icons, Badges, Row Highlighting.
- **Selection:** Colors and Icons.
- **Notes:** Direct visual mapping for each priority level, with special emphasis on "Diamante".

### 4. Flashcard Editing
- **Options presented:** Markdown, Plain Text, Structured Fields.
- **Selection:** Markdown.
- **User Constraint:** Must allow images embedded in the text (URLs/Base64).
- **Notes:** Implementation will use a Markdown library with image support.
