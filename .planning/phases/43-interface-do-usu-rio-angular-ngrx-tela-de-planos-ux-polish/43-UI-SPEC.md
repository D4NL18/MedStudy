## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--bg-default)` | Background, surfaces |
| Secondary (30%) | `var(--surface-color)` | Cards, sidebar, nav, modal backgrounds |
| Accent (10%) | `var(--primary-color)` | Primary CTA buttons, active states, progress indicators |
| Destructive | `var(--danger-color)` | Destructive actions, expired subscription banner |
| Warning | `var(--warning-color)` | Expiring soon banner |

Accent reserved for: Primary calls to action ("Assinar", "Já Paguei") and active links. Never used for destructive or background elements.
