## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (`--spacing-xs`) |
| sm | 8px | Compact element spacing (`--spacing-sm`) |
| md | 16px | Default element spacing (`--spacing-md`) |
| lg | 24px | Section padding (`--spacing-lg`) |
| xl | 32px | Layout gaps (`--spacing-xl`) |
| 2xl | 48px | Major section breaks (`--spacing-2xl`) |
| 3xl | 64px | Page-level spacing (`--spacing-3xl`) |

Exceptions: none

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 600 | 1.4 |
| Heading | 24px | 600 | 1.2 |
| Display | 32px | 700 | 1.2 |

*(Note: Utilizing existing global classes from `styles.scss` for headings)*

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--color-bg)` | Background, main surfaces |
| Secondary (30%) | `var(--color-surface)` | Cards, glass panels, sidebars |
| Accent (10%) | `var(--color-accent)` | Primary buttons, active states, icons |
| Destructive | `#EF4444` | Destructive actions, errors, delete buttons |

Accent reserved for: Primary interactive elements (buttons, toggles) and indicators of state selection.

---

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Padrão: Confirmar / Salvar |
| Empty state heading | Nenhum resultado encontrado |
| Empty state body | Não existem dados para exibir nesta visão. |
| Error state | Ocorreu um erro ao carregar o componente. |
| Destructive confirmation | Excluir: Tem certeza que deseja remover este item? |

*(Note: As this phase focuses on generic component extraction (`<app-button>`, `<app-card>`), copywriting applies to defaults and fallback states.)*
