**Description:**

</read_first>

<action>
1. **`button.component.ts`**:
   - Add `@Input() customClass = '';`.
   - Update `get buttonClasses()` to append `this.customClass`.
2. **`modal-layout.component.ts`**:
   - Add `@Input() useDefaultFooter = true;`.
   - Add `@Input() fullScreenOverlay = false;`.
3. **`modal-layout.component.html`**:
   - Create an `<ng-template #modalContent>` containing the modal header, content, and footer logic.

</action>
